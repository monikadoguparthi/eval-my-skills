# 🚀 HireSense AI

> **Turn practice into performance.**

HireSense AI is an AI-powered mock interview platform that helps students and job seekers prepare for technical and HR interviews through realistic interview simulations, instant AI-powered feedback, and voice-enabled responses.

---

## 🌐 Live Demo

🔗 https://eval-my-skills.vercel.app

---

## ✨ Features

### 🎯 AI Mock Interviews
- Role-based interview generation
- Technical & HR interview modes
- Multiple difficulty levels (Easy, Medium, Hard)
- Configurable interview duration

### 🎤 Voice-to-Text
- Browser Speech Recognition API
- Real-time speech transcription
- Hands-free answering experience
- Automatic silence detection
- Microphone permission handling

### 🤖 AI Evaluation
- Technical knowledge assessment
- Communication analysis
- Confidence evaluation
- Personalized improvement suggestions
- Constructive interview feedback

### 📊 Performance Dashboard
- Overall interview score
- Technical score
- Communication score
- Confidence score
- Detailed strengths & weaknesses
- Progress tracking

### 🔐 Authentication
- Google Sign-In
- Email & Password Login
- Secure user authentication
- Protected routes

### 🎨 Modern UI
- Responsive design
- Dark & Light mode
- Smooth animations
- Mobile-friendly interface
- Professional SaaS-inspired layout

# 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/monikadoguparthi/eval-my-skills.git
```

### Navigate to the project

```bash
cd eval-my-skills
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open:

```
http://localhost:5173
```

---

# 🔐 Environment Variables

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_OPENAI_API_KEY=your_openai_api_key
```
